浅谈“知识管理”、“工具收集”以及“开发环境的整理”

发表于 2016-09-10   |   分类于 Tool   |     |   阅读次数

工欲善其事，必先利其器！

前言

蓦然回首，发觉自己已经好久没写博客了，可能是源于博客更新起来费时费力，也确实，从构思到发布需要一大把时间，有些主题也是被大神们写的太多了，也就不重复造轮子了，也或许是自己找到了更好的知识管理系统，总之今天写这个博客也算是对前段时间学习的总结，其中涉及到的工具，插件也便于工作以后查找使用！

知识管理

俗话说：“好记性不如烂笔头”，可见学习过程中记笔记的重要性，然而如何记笔记才能方便以后的查找，说说我的历程，刚开始学习前端知识，是采用传统的记笔记方式，对，没错，记在本子上，代码之类的还是要在开发工具中运行试试效果的，之后随着笔记做的越来越多，你再想查询一个知识点可就难了，因为你很难记清楚哪个知识点在哪一页啊，有翻笔记本的时间在网上恐怕早就搜到了！可见知识管理的重要性，有好的工具才能事半功倍，加快效率！后来发现了云笔记之类的软件，顾名思义，这类工具就是把笔记写在云端，把笔记放在网上自己的空间里，再查找时可以根据标签分类快速查找到！当找到这些工具是，我的内心里是非常欣喜的，犹如发现了新大陆的感觉，不禁又感觉自己以前做笔记的方式是多么low，但是也不是说纸质笔记就不好，各有千秋，纸质笔记与云笔记相辅相成，才能最大化为知识管理做出最大贡献！下面我就来安利一下最近发现的好用的知识管理类工具:

  • 知识收集类

    • Pocket【看到一个好文章而暂时没时间阅读，关闭后说不定又找不到了，怎么办怎么办怎么办？用这个工具啊，他可以把你想看还没来得及看的网页存到Pocekt上，有空再去看咯，不得不说，这款产品的出发点还是很棒的】
    • 收趣云标签【国内版的Pocket，界面之类也还够小清新，但是我还是偏爱Pocket】

  • 知识管理类

    • 为知笔记【更适合国人使用的云笔记
      多级目录、多级标签、Markdown、无限存贮空间，不过之所以我最终选择了这一款云笔记，最大原因还是他免费啊，而且markdown支持的比较好，设置多级标签及分类后很容易管理】
    • 印象笔记【号称“记住一切
      现代生活喧嚣繁杂，印象笔记帮你轻松简化一切，让生活更美好，让工作更省力”，不过确实也有这个实力，但是就是太抠了，一个月就给60M上传流量，那能够吗？而且还没有内置markdown】
    • 有道云笔记【网易出品，支持多种Office文档，办公室随身携带】
    • Simplenote【一款很小众的云笔记，正如其名非常简单，小巧，适合日常记事，正如周倩如大神在《非你莫属》上诠释的那样，什么是一款好的互联网产品，“你要把互联网用户想象成智商非常低，脾气非常差，超级没有耐心，而且超级小气，不肯出一分钱的人”只要把这种人给伺候好了，这样的产品才会容易获得成功吧！】

上述这些软件，pc端移动端都有！由于是保存在云端，何时何地采用不同终端依然可以查看！

开发环境整理

这么多工具、插件会不会很乱，答案只有一个，你如果不学着去整理，那么肯定是会乱的。

日常生活中我也是一个爱整理的人，每隔一段是时间就会整理自己的书桌，衣柜，整理过后，自己也会觉得心情大好！

如何整理

电脑桌面

电脑桌面整理，不要把什么乱七八糟的图标全部都摆在桌面上，因为看上去太乱了，可以把软件放到开始目录内，我使用的是window10系统，具体操作参考如何优雅地使用 Windows 10 ?

电脑性能

原来想在网上买块固态内存自己换的，转念一想，那样的话，所以软件都要重装，就尝试一系列别的方法去解决电脑的卡慢，比如磁盘清理，增加虚拟内存，去除一些不必要的动画效果…以及更有逼格的关机方式SlideToShutDown,滑下来即可关机

浏览器

浏览器整理，书签要分类，配合云笔记和Pocket使用，不要把网页都设置成书签还不分类，那时间长了，书签将臃肿不堪！

云笔记

云笔记整理，来不及看的页面，想存到Pocket,看完之后，要重要知识点的保存到云笔记，云笔记，分类以及标签要分级，方便以后的查找

移动端

顺便安利一些手机端好用的软件:

  • Via浏览器【轻快简洁，200多kb安装包，一用倾心，你值得拥有】
  • since【一款简单记录天数的计数器】
  • 网易云音乐【根据你的听歌偏好，每天为你推荐20首歌，推荐的都还不错哦，目前每天都会打开网易云音乐来听歌】
  • Smooth【一个简单的新浪微博客户端，刷刷微博还好，但是有不少接口不允许使用】
  • 有道词典【这个也是基本上每天都会用到，英语单词查询必备】
